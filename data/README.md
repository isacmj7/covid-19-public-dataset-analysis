# Data Folder

Download the data files from **COVID19India.org** (direct CSV downloads):

**Source:** https://data.covid19india.org/

Required files (right-click and "Save As"):

1. **State-wise Time Series Data**
   - URL: https://data.covid19india.org/csv/latest/states.csv
   - Save as: `states.csv`

2. **India Daily Time Series**
   - URL: https://data.covid19india.org/csv/latest/case_time_series.csv
   - Save as: `case_time_series.csv`

3. **State-wise Daily Data**
   - URL: https://data.covid19india.org/csv/latest/state_wise_daily.csv
   - Save as: `state_wise_daily.csv`

4. **State-wise Cumulative Data**
   - URL: https://data.covid19india.org/csv/latest/state_wise.csv
   - Save as: `state_wise.csv`

Place all CSV files in this folder before running the notebook.
