## How to Run

1. Install packages:
   ```
   pip install -r requirements.txt
   ```

2. Download dataset from COVID19India.org:
   https://data.covid19india.org/

3. Place all CSV files in the `data/` folder (see data/README.md for file names)

4. Open notebook:
   ```
   jupyter notebook notebooks/01_covid19_india_analysis.ipynb
   ```

5. Run all cells (Cell â†’ Run All)
